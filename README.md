# InAmigos AI Assistant

## Overview

InAmigos AI Assistant is an AI-powered chatbot developed for InAmigos Foundation using Google's Gemini API and Streamlit.

The chatbot answers user queries based only on information extracted from the official InAmigos Foundation website.

## Features

* NGO-specific question answering
* Prompt-engineered response control
* Multi-turn conversation support
* Context-aware responses
* Fallback handling for unknown questions
* Conversation logging
* Streamlit-based user interface

## Technologies Used

* Python
* Gemini API
* Streamlit
* Prompt Engineering
* Web Scraping
* BeautifulSoup
* dotenv

## Project Workflow

1. Scrape website content
2. Extract useful text
3. Store NGO knowledge base
4. Send user query to Gemini
5. Generate response using prompt engineering
6. Display response in Streamlit interface

## Example Questions

* Who founded InAmigos Foundation?
* What is Project Udaan?
* What is Project Jeev?
* How can I contact the NGO?
